前言

首先非常感谢大家访问我的 Blog，这是我搭建的第一个博客性质的网站。使用的是 Github Pages + Hexo 的形式搭建，前前后后几分钟就能搭建完成了叭，现在把它记录下来，将来也许会是一种怀念叭。

须知

Github

搭建一个这样的 Blog 首先你要知道什么是 Github 和 Github Pages，你可以简单理解为咱们没有服务器，咱将页面托管到了 Github 上。

Github Pages

Github Pages 可以被认为是用户编写的、托管在 github 上的静态网页。使用 Github Pages 可以为你提供一个免费的服务器，免去了自己搭建服务器和写数据库的麻烦。此外还可以绑定自己的域名。

这是 Github Pages 的官网 Github Pages

Hexo

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。
